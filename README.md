# ProBuild - Premium Construction Company Template

A modern, fully responsive Next.js website template for construction and building companies. Built with TypeScript, Tailwind CSS, GSAP animations, and Framer Motion.

## Features

- **4 Color Themes** - Navy & Gold, Forest & Copper, Charcoal & Blush, Teal & Coral
- **Smooth Animations** - GSAP ScrollTrigger animations with Lenis smooth scrolling
- **5 Pages** - Home, Services, Portfolio, About, Contact
- **Multi-Step Contact Form** - Wizard-style form with project details
- **Portfolio Gallery** - Filterable gallery with lightbox viewer
- **Testimonial Carousel** - Swiper.js powered testimonial slider
- **Fully Responsive** - Mobile-first design approach
- **Image Optimized** - Next.js Image component with Unsplash images

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 3.4
- GSAP + ScrollTrigger
- Framer Motion
- Swiper.js
- Lucide React Icons
- Lenis Smooth Scroll

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` - Home (Hero, Services, About, Gallery, Testimonials, CTA)
- `/services` - Detailed service pages (Custom Homes, Commercial, Renovations, etc.)
- `/gallery` - Filterable project portfolio with lightbox
- `/about` - Company story, team, values, timeline
- `/contact` - Multi-step contact wizard with project details
